## Setup

The following commands will get you up and running:

### Install

```bash
$ npm install
```

### Run

```bash
$ npm run serve
```

### Build

Build files will be placed in `./dist`.

```bash
$ npm run build
```

To serve static build files, I recommend using [serve](https://github.com/vercel/serve#readme).

```bash
$ serve dist
```
